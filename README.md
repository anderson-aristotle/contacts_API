# E-Contacts!

A online contacts database where user are able to create, read, update, and delete a contact resource.

## Getting Started

Go to https://github.com/anderson-aristotle/contacts_client to explore client code.

## Technologies used

+ Bootstrap
+ CSS
+ Handlebars
+ HTML
+ Javascript
+ jQuery
+ Ruby on Rails
